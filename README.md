# SEAPREAD SHEETS

This is **Seapread Sheets**. try saying it very fast. A frontend task to mimic some functionalities of a spreadsheet.


# Local preview

 1. Clone repo
 2. install dependencies via `npm intall`
 3. Spin up the dev server via `npm run start`
 
# Online Demo
